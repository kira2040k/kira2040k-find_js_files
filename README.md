# usage:
```
usage: main.py [-h] [-u url] [-f file] [-c] [-a] [-usernames] [-passwords]

find js files in urls

optional arguments:
  -h, --help  show this help message and exit
  -u url      target url
  -f file     target file
  -c          print comments
  -a          include jquery and CDN etc...
  -usernames  find usernames
  -passwords  find passwords
```
